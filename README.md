# 転路 - サイコロ戦略ボードゲーム公式サイト

このプロジェクトは、新感覚ボードゲーム『転路』を紹介するためのWebページです。

## 特徴
- 転路の基本ルールと魅力を紹介
- ゲームのダウンロードリンクを提供

## ページ構成
1. **トップページ**: ゲームの概要を説明
2. **特徴セクション**: ゲームの魅力をリストアップ
3. **ダウンロードセクション**: ゲームへのリンクを表示

## 使用技術
- HTML5
- CSS3
- JavaScript

## 公開方法
1. このリポジトリをGitHubにプッシュします。
2. GitHub Pagesを有効化して公開します。
